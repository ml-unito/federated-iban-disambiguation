# Intesa NER
This repository contains the following directory:
- `./synthetic-data-generator/`: contains the script for generating synthetic data;
- `./politecnico-code-reproduction/`: contains the script with the reproduction of the code described in the Politecnico di Torino article for solving the problem;
- `./GraphNeuralNetworks/`: contains the scripts for solving the problem using Graph Neural Network
- `./intra-accounts-disambiguation/`: contains the scripts for solving the problem using Character Bert and Spectrum Kernel. Files are available for both centralized and federated training with Fluke.
- `./flower-mlp/`: contains scripts for federating the Spectrum Kernel model with MLP using Flower.
- `./flower-bertmlp/`: contains scripts for federating Character Bert using Flower.
